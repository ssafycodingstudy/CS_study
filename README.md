# CS 스터디

## 📌 스터디 방식
- https://github.com/gyoogle/tech-interview-for-developer 참고
- 주제는 3~4개씩 묶어서 발표 준비 (자료 양에 따라 유동적으로 조정)
- 모두 매주 목요일까지 발표 준비
- 발표는 랜덤 1명 (2주연속 걸리면 한번 면제)
- 자료 참고해서 준비하되 추가적인 내용들도 가져오기
- 각자 질문 1개씩 하기 (중복 가능성 있기때문에 여유있게 2~3개 준비해오기)
- 발표 자료는 모두 Git에 올리기
