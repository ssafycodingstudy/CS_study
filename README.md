# CS 스터디

## 📌 스터디 방식
- https://github.com/gyoogle/tech-interview-for-developer 참고
- 주제는 2개씩 묶어서 발표 준비
- 모두 매주 목요일까지 발표 준비
- 발표는 랜덤 2명
- 2주 연속 발표시 한번 pass
- 자료 참고해서 준비하되 추가적인 내용들도 가져오기
- 발표 자료는 모두 Git에 올리기
- 벌금
  - 발표자료 미제출시 1개 주제당 ₩ 2,000
  - 발표날(목요일) 결석 시 ₩ 5,000
  - 10분 이상 지각 시 10분당 ₩ 1,000
